# Clone the project  
~~~
- $ git clone git@gitlab.com:christianesr/final-project.git  
~~~
## Install dependencies  
~~~
- $ composer install
~~~

## Start the server  
~~~
- $ php artisan serve
~~~
# Run migrations
~~~
- $ php artisan migrate
~~~

# show relations between models
~~~
- $ php artisan model:show Api/Candidate
~~~
